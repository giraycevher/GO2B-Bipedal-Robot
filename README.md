# GO2B-Bipedal-Robot
12 DoF LIPM ZMP Based Preview Control

A high-fidelity bipedal robotics platform featuring ZMP-based trajectory planning and MuJoCo simulation.

Project Overview
GO2B is an R&D platform designed to validate control architectures for bipedal locomotion. This repository demonstrates the integration of MPC-based Preview Control, Inverse Kinematics (IK), and real-time sensor feedback within a MuJoCo physics environment.

Key Capabilities
MPC Trajectory Planning: Robust locomotion planning using ZMP preview control.

Sim-to-Real Architecture: Python-based control loops optimized for hardware interfacing.

Mechanical Design: 12-DOF bipedal leg system optimized for CNC manufacturing.

Acknowledgements
The core ZMP Preview Control mathematical framework is adapted from Eko Rudiawan's ZMP-Preview-Control-WPG repository. All IK Code based on Introduction to Humanoid Robotics (Kajita ,2014)
MuJoCo physical modeling, and GenAI system integrations were developed as part of the GO2B project.
